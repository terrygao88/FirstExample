Smart Umbrella System


This project is to design a smart umbrella system and it can be attached to a wheelchair, which can block the sunlight for the wheelchair user. Sunlight is necessary to a person’s life; however, excessive sunlight could potentially cause a lot of problems, especially to the wheelchairs users. Most electric wheelchairs in the current market are giving people freedom to move around without manual power, but those wheelchairs don’t have features such as a sun tracking umbrella. My project will solve this problem. The solution is just simply implementing a sun tracker algorithm to Arduino Mega board with two servos, and the servos will be used to control the umbrella. We can attach the whole system to the wheelchair; and the umbrella will automatically adjust its position depending on the position of the sun, thus blocking sunlight for the users.

Temperature Station System and Its Application


Two Arduino Uno boards are used for this project, the first Arduino board will be used as a yard sprinkler system controller, the second Arduino board is used to get all the sensor data and sending them to the first microcontroller. In addition, there will be an android phone application that can get all the sensor reading from Arduino board via Bluetooth communication. If a person wants to check the surrounding temperature information, she can connect her phone to the Arduino board to get all the information. The android application will also give a “tip” to the user, advising a suitable daily activity for the user.

Student Record System: JavaFX Project


This JAVA application is designed for record student information and student learning. When logged in as an administrator, it is able to query student information from database, display student information in a dialog, add, delete and edit student entries and save them to database. It also gives users the functionality to print the displayed student information. when logged in as a student, a user is able to play the lectures videos. Depend on the course that she enrolled in, different videos will be displayed in the student learning window. A simple playback tool with progress bar is available and giving the student freedom to choose where she wants to play the video.
